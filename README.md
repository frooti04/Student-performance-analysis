# Student-performance-analysis
Students analysis using Excel
## Objective
To analyze student marks and evaluate academic performance
## Dataset
Student ID, Name, Gender, Math, English, Science, Total, Percentage, Grade
## Tools Used
Microsoft Excel
## Analysis Performed
-Total and Percentage calculation
-Grade classification using formulas
-Gender wise performance analysis
-Subject wise average marks
## Dashboard
Created dashboard to visualize performance trends
